# 🚀 Hyprland Cheatsheet - TheGirlPuma

**Super = Windows-Taste**

**Wichtig:**
- Super + D          → App-Menü
- Super + Q          → Fenster schließen
- Super + Linksklick → Fenster verschieben
- Super + Rechtsklick→ Fenstergröße ändern
- Super + Shift + E  → Hyprland beenden
- ❔                 → Dieses Cheatsheet
- ⏻                 → Power Menu (Logout, Shutdown)
